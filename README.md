# Social Media App

# Description

It's an API for a web application for a social network where users can express their ideas, respond to those of friends, and build a friend list.

# Technology

* Javascript
* Node.js
* Express.js
* MongoDB
* Mongoose
* ThreeJS 
* React Three Fiber 
* TailwindCSS 
* Framer Motion 

# Installation

npm install

# Usuage 

npm run dev
